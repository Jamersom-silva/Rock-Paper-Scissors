# 🎮 Rock Paper Scissors

A modern web-based Rock Paper Scissors game built with **Flask**, featuring real-time scoring, match history, and a responsive UI.

🌐 Live Demo: https://rock-paper-scissors.onrender.com

---

## ✨ Features

- 🎮 Classic Rock Paper Scissors gameplay
- 🤖 Play against a random computer AI
- 🏆 Real-time score tracking
- 📊 Match history for all rounds
- 🎯 Customizable rounds (1–20)
- 📱 Fully responsive design (mobile + desktop)

---

## 🛠 Tech Stack

- Python (Flask)
- HTML5
- CSS3
- Gunicorn
- Render (Deployment)

---

## 🧠 How it works

Player selects Rock, Paper, or Scissors → Computer randomly selects → Winner is calculated instantly → Score updates → Game ends after selected rounds.

---

## 📁 Project Structure

- app.py → Main Flask application
- templates/ → HTML pages
- static/ → CSS and assets
- requirements.txt → Python dependencies
